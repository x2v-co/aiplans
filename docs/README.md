# Docs

This directory contains project documentation.

## Directory Structure

```
docs/
├── README.md           # This file
└── archive/           # Archived task completion notes and historical docs
```

## Root Documentation

The following documentation files are kept in the project root:

- **README.md** - Project overview and getting started
- **CLAUDE.md** - Project instructions for Claude Code AI assistant
- **DEVELOPMENT_PLAN.md** - Development roadmap and planned features
- **MRD.md** - Product Requirements Document

## Archive

The `archive/` directory contains historical documentation from completed tasks and development milestones. These files are kept for reference but are no longer actively maintained.

Topics in archive include:
- Scraper implementation notes
- i18n migration documentation
- Currency and pricing fixes
- Feature implementation logs
- Bug fix notes
